# FUTURE_CS_01
Vulnerability Assessment Report for a Live Website using OWASP ZAP, Nmap, and Kali Linux as part of the Future Interns Cyber Security Internship.
## 📌 Project Description

This project presents a comprehensive Vulnerability Assessment conducted on a live web application using industry-standard cybersecurity tools such as OWASP ZAP, Nmap, and Kali Linux. The assessment was performed as part of the Future Interns Cyber Security Internship program.

The objective of the project was to identify potential security vulnerabilities, analyze exposed services, evaluate website security posture, and recommend remediation measures to improve overall security. The assessment included reconnaissance, port scanning, vulnerability detection, risk analysis, and security reporting.

### 🛠 Tools Used

* OWASP ZAP
* Nmap
* Kali Linux
* Browser Developer Tools

### 🎯 Key Activities

* Website reconnaissance and information gathering
* Network and port scanning
* Vulnerability identification and analysis
* Security risk assessment
* Documentation and reporting of findings

### 📈 Outcome

Successfully identified potential security weaknesses and generated a detailed vulnerability assessment report with recommendations to enhance website security and reduce attack surface.
